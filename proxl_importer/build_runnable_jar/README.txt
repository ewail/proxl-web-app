
How to create a Proxl Importer Runnable Jar


Run ant__create_runnable_jar.xml with ant:
  (in the same directory this file is in )

    ant -f ant__create_runnable_jar.xml


The updated file will be in the "deploy" folder.



What is changed:

A runnable JAR is created:

	A manifest with the Java main class used in the importer
